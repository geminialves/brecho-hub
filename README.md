# 🛍️ Brechó Virtual

Uma plataforma web para compra e venda de roupas, calçados e acessórios usados, promovendo a moda sustentável e incentivando a economia circular.

---

## 📖 Sobre o Projeto

O **Brechó Virtual** foi desenvolvido com o objetivo de conectar compradores e vendedores em um ambiente seguro, intuitivo e acessível.

A plataforma permite que usuários anunciem produtos usados, pesquisem itens disponíveis, realizem compras e acompanhem todo o histórico de transações.

---

## 🎯 Objetivos

* Incentivar a reutilização de roupas e acessórios.
* Promover a economia circular.
* Facilitar a compra e venda de produtos usados.
* Aumentar a visibilidade dos vendedores.
* Proporcionar uma experiência simples e segura para os usuários.

---

## 👥 Público-Alvo

* Pessoas interessadas em moda sustentável.
* Usuários que desejam vender roupas, calçados e acessórios usados.
* Consumidores em busca de produtos de qualidade com preços acessíveis.

## 🚀 Como Executar o Projeto

### Pré-requisitos

Antes de iniciar, certifique-se de possuir:

* Node.js instalado
* npm instalado
* Git instalado

---

### Clone o Repositório

```bash
git clone https://github.com/seu-usuario/brecho-virtual.git

cd brecho-virtual
```

---

### Instale as Dependências

```bash
npm install
```

---

### Configure as Variáveis de Ambiente

Crie um arquivo chamado `.env.local` na raiz do projeto:

---

### Executar em Desenvolvimento

```bash
npm run dev
```

Após iniciar a aplicação, acesse:

```txt
http://localhost:3000
```

---

### Gerar Build de Produção

```bash
npm run build
```

---

### Executar em Produção

```bash
npm start
```

## ✅ Definition of Done (DoD)

Uma funcionalidade será considerada concluída quando:

* Desenvolvimento finalizado;
* Código revisado;
* Testes executados e aprovados;
* Critérios de aceitação atendidos;
